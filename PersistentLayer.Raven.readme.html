Welcome to PersistentLayer.Raven

PersistentLayer.Raven is a component which implements contracts defined on PersistentLayer component, and its underlaying implementation is based on NHibernate.

Latest Version

The quickest way to get the latest release of PersistentLayer.Raven is to add it to your project using NuGet (http://nuget.org/List/Packages/PersistentLayer.Raven).
